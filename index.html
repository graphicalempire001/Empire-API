🚀 Custom Endpoints & Services

🎥 Video Downloader

Fast, robust download of media URLs using a managed yt-dlp backend wrapper.

GET /api/dl?url={URL}&type={mp4|mp3}

🎨 AI Image Generator

Generates high-fidelity visual representations from textual query prompts.

GET /api/ai/draw?q={PROMPT}

⚽ Sports Live Tracker

Retrieves matches, league standings, and live score updates.

GET /api/sports/live

☁️ Weather Forecast

Accurate, localized conditions and upcoming multi-day prognoses.

GET /api/weather?q={CITY}

📄 PDF maker

Compiles styled document bodies into full, standard PDFs on the fly.

POST /api/pdf/create { title, content }

🔍 OCR Engine

Performs optical character recognition directly on external image links.

POST /api/ocr { imageUrl }

📦 Bot Plugin Integration Template

Install the API wrapper directly in your Empire-MD repository under lib/empireApi.js:

const axios = require('axios');
const API_BASE = "https://your-deployed-instance.com/api";

const EmpireAPI = {
    downloadMedia: (url, type) => axios.get(`${API_BASE}/dl`, { params: { url, type } }),
    aiImagine: (prompt) => axios.get(`${API_BASE}/ai/draw`, { params: { q: prompt } }),
    getWeather: (city) => axios.get(`${API_BASE}/weather`, { params: { q: city } })
};

module.exports = EmpireAPI;
